Установка на телефон

      
Открой HTML-файл в Chrome, нажми меню (⋮) → «Добавить на главный экран». Приложение появится как иконка и будет открываться без адресной строки. Интернет для работы не нужен.

    


    

      
Шапка

      
⬆ Импорт — загрузить ранее сохранённый JSON-файл резервной копии. Текущие данные будут заменены.

      
⬇ Экспорт — сохранить все данные в JSON-файл на устройство. Делай периодически: данные хранятся в памяти браузера и пропадут при очистке кэша.

      
? — эта справка.

    


    

      
Вкладка Прайс

      
Справочник всех видов работ с ценой за единицу. Отсюда выбираются позиции при создании обращения и при заполнении причины вызова.

      
Строка поиска фильтрует список по названию работы — курсор не сбрасывается при вводе. Кнопка + добавляет новую работу. ✏️ редактирует, 🗑️ удаляет.

    


    

      
Вкладка Клиенты

      
Свёрнутая карточка показывает название, множитель ×1.5, телефон первого контакта и адрес. Тап раскрывает карточку.

      
Кнопки управления в правом нижнем углу:

      
+ — добавить нового клиента.

      ⇅ — свернуть или развернуть все карточки разом. Если все раскрыты — сворачивает, иначе разворачивает. Учитывает текущий фильтр и поиск.

      
Фильтры под строкой поиска:

      

        Все — полный список.

        🟡 В работе — только клиенты с незавершёнными обращениями.

        🔴 Неопл. — только клиенты с неоплаченными обращениями. У свёрнутых карточек отображается сумма долга и количество обращений. Над нижним меню — общая сумма по всем видимым клиентам. Кнопки ⇅ и + поднимаются над этой полосой.

        ✅ Сделано — только клиенты с оплаченными обращениями.

        При выборе фильтра карточки раскрываются автоматически и показывают только подходящие записи. На кнопках отображается количество таких обращений.
      

      
Поиск работает по названию организации, адресу, именам, должностям и телефонам контактных лиц.

    


    

      
Настройки

      
Открываются кнопкой ⚙️ в нижнем меню. Настройки хранятся отдельно от базы данных и не затрагиваются при импорте/экспорте.

      
Показывать множитель — управляет видимостью бейджа ×1.5 в шапке карточки клиента. Удобно отключить, если множитель у всех клиентов одинаковый.

      
Шапка счёта — текст, который появляется перед счётом при отправке: реквизиты, имя специалиста, телефон и т.п.

      
Подвал счёта — текст после итоговой суммы: реквизиты для оплаты, подпись, пожелания.

      
Скрытые клиенты — список всех клиентов с галочками. Снятые клиенты не отображаются в основном списке, их обращения не попадают в счётчики фильтров и не учитываются в общей сумме «Не оплачено». Галочку можно вернуть в любой момент — данные не удаляются.

    


    

      
Счёт клиенту

      
Кнопка «📄 Счёт» появляется в раскрытой карточке клиента только при активном фильтре 🔴 Неопл. — когда есть что выставлять.

      
Открывается модальное окно со списком всех неоплаченных обращений. Каждая работа показана с количеством и суммой — галочкой можно исключить любую позицию из счёта. Итог пересчитывается в реальном времени.

      
Выезды сгруппированы по дате и причине. Шапка и подвал добавляются из настроек. Формат текста:

      
Счёт · Название клиента
━━━━━━━━━━━━━━━━━━━━━━━━

21-04-2025 · Замена блока питания
  Блок питания ATX — 1 шт — 2 500 ₽
  Работа по замене — 1 шт — 800 ₽

━━━━━━━━━━━━━━━━━━━━━━━━
Итого: 3 300 ₽

      
📤 Поделиться — открывает системный диалог выбора приложения. Текст вставляется в новое сообщение или письмо — ничего не отправляется автоматически.

      📋 Скопировать — копирует текст в буфер обмена. Если буфер недоступен (небезопасный контекст) — показывается текстовое поле для ручного копирования.

      
Шапку и подвал счёта (реквизиты, подпись, реквизиты оплаты) можно настроить в разделе ⚙️ Настройки.

    


    

      
Карточка клиента

      
Название — имя или название организации.

      
Адрес — адрес объекта.

      
Множитель — коэффициент наценки, применяется ко всем работам этого клиента по умолчанию. Например, 1.5 означает +50% к прайсовой цене.

      
Контактные лица — можно добавить любое количество. Каждый контакт: имя, должность (директор, бухгалтер, начальник отдела...), телефон. В раскрытой карточке они показываются списком: должность · имя · телефон.

      
Заметки — произвольный текст о клиенте.

    


    

      
Обращение — создание и редактирование

      
Кнопка «+ Добавить обращение» внутри карточки открывает полноэкранный редактор. Тап на существующее обращение открывает его для редактирования.

      
Статусы обращения:

      

        В работе — янтарный фон, работа ещё не завершена.

        Не оплачено — красный фон, выполнено но не оплачено.

        Оплачено — без цвета, закрытая история.
      

      
Причина вызова — текстовое поле. Кнопка «📋 Из прайса» открывает список работ: тап вставляет название в поле. Если поле уже заполнено — текст добавляется через запятую. Можно вставить несколько позиций подряд и затем отредактировать вручную.

    


    

      
Работы и расчёт суммы

      
Каждая строка работы содержит три поля: Кол-во, Множитель, Итого ₽.

      
Автоматический режим (поле итого синее): итого = цена из прайса × кол-во × множитель. Множитель наследуется от клиента, но для каждой строки его можно изменить отдельно.

      
Ручной ввод: если вписать сумму прямо в поле Итого, оно становится янтарным. Множитель тоже становится янтарным со значением 0 и подписью «РУЧ. ВВОД» — сигнал, что автоматический расчёт отключён для этой строки.

      
Чтобы вернуться к автоматическому расчёту — вписать любое ненулевое значение в поле Множитель. Поля вернутся к синему цвету.

      
Общий итог внизу — сумма всех строк, с учётом ручных значений.

    


    

      
Резервное копирование

      
В браузере: данные хранятся в localStorage. При очистке кэша браузера данные пропадут. Рекомендуется периодически делать экспорт (кнопка ⬇ в шапке).

      
В APK-версии: данные автоматически сохраняются во внутреннюю папку приложения (файл ito_data.json) при каждом изменении и при сворачивании. Папка не видна в файловом менеджере, но данные не зависят от очистки кэша браузера и сохраняются между запусками. Дополнительно данные дублируются в localStorage.

      
Кнопка ⬇ Экспорт создаёт файл резервной копии с датой в имени и открывает диалог «Поделиться» — можно отправить в Telegram, на почту, сохранить в Google Drive или Downloads. Это единственный способ получить файл с данными на руки.

      
Кнопка ⬆ Импорт — загрузить ранее сохранённый файл. Текущие данные при импорте заменяются полностью. Сразу после импорта появляется уведомление со ссылкой «Отменить» — в течение 8 секунд можно откатить замену и вернуть предыдущее состояние базы.

    


    

      
Настройки

      
Открываются кнопкой ⚙️ в нижнем меню. Закрываются стрелкой ‹ или системной кнопкой «Назад».

      
Показывать множитель — включает или скрывает бейдж ×1.5 в шапке карточки клиента. Удобно отключить, если множитель у всех клиентов одинаковый.

      
Шапка счёта — текст, который добавляется перед названием клиента при формировании счёта. Подходит для реквизитов: имя ИП, ИНН, телефон.

      
Подвал счёта — текст, который добавляется после итоговой суммы. Подходит для реквизитов оплаты, подписи, благодарности.

      
Скрытые клиенты — список всех клиентов с галочками. Снятая галочка убирает клиента из основного списка, его обращения не учитываются в счётчиках фильтров и в полосе общей суммы. Данные клиента и его история сохраняются — можно вернуть галочку в любой момент.

      
Скрытие клиентов удобно для временно неактивных контрагентов или для разделения рабочих и архивных записей без удаления.
